# 交易条件判断规则

> 本文档是所有交易判断逻辑的唯一来源。**修改代码前，必须先修改本文档。**
>
> 工作流：修改本文档中的规则描述 → 更新代码 → 运行批量验证 → 根据结果继续调优。

---

## 术语表

### 运行与调整

市场走势就是无数个**运行**和**调整**的持续组合。

- **运行** = 释放。价格朝一个方向持续推进，消耗动能。
  - 特征：N字形、连续性、强势动能、斜率越大消耗越快
  - 好比呼气：在消耗积累的能量

- **调整** = 蓄力。价格在一个区间内反复震荡，积累筹码。
  - 特征：W/M形、反复性、弱势动能、斜率与运行方向相反
  - 好比吸气：在积累新的能量

运行和调整交替出现是趋势健康的保障。本系统的核心理念：**不在释放中参与，尽量在蓄力充足后参与**。

### A段

A段是筹码积累充足后的**第一段运行**。识别口诀：找到最后一段调整，锁定最后一段运行，被逆转后确立新的运行方向。这正是我们六维分析要捕捉的机会。

### 关键位

- **平台位**：盘整过程中价格反复触碰的水平价位，好比"中场休息时搭的台子"
- **颈线位**：趋势反转形态（头肩形态）中的关键位，代表"趋势已经改变"

## 总体框架

我们用六个维度来判断一只股票是否具备开仓条件：

| 序号 | 维度 | 简称 | 核心问题 |
|------|------|------|----------|
| 1 | 平台位 | PT | 调整区间内是否形成了被反复验证的关键价位？ |
| 2 | 独立结构 | DL | 是否经历了足够长时间的横盘调整？（由PT反推确定） |
| 3 | 轮廓质量 | LK | 调整的形态是否工整、波浪规则、边界平滑？ |
| 4 | 释放级别 | SF | 调整期间整体是否保持水平，还是已经向某个方向偏移释放了动能？ |
| 5 | 统一区间 | TY | 调整末尾是否出现了K线越来越小的挤压收敛？ |
| 6 | 动能 | DN | 是否出现了有力量的突破K线？ |

**执行顺序**: PT → DL → LK → SF → TY → DN

- 先找PT位候选（至少2次测试即可识别候选价位，确定DL结构边界并纳入观察），但PT位合格需≥3次测试
- 找不出PT位候选（不足2次测试）→ 不能形成有效DL
- DL ≥90根通过，<90根继续分析后续维度但标注"结构未成熟，待观察"
- 每个维度独立评分（DL/PT/LK/TY为S/A/B/C，SF为1st/2nd/3rd，TY和DN可为待定）
- **不做加权打分**，不做综合评级，六个维度的结果直接展示，由人来综合判断

## 第一维度：独立结构 (DL)

### 判断目标

确认股票正在经历一段**水平方向的调整**（盘整），而不是处于单边运行中。调整时间越长，说明筹码交换越充分，后续突破的可信度越高。

### 判断规则

1. DL结构的边界由PT位反推确定：至少2次测试可识别候选价位并确定结构边界（PT评分合格需≥3次测试）。不足2次测试 → 不能形成有效DL
2. 判断"走平"的标准：每20根K线为一组，组内价格变化速率不超过每根K线0.3%
3. 允许调整段中间夹杂少量波动（最多容忍15组不平的窗口），只要整体仍然呈现横盘特征
4. **结构边界截断规则**：遇到以下情况时，结构区间应在此处截断，不再向前扩展：
   - 明显的趋势段（连续大幅上涨或下跌，斜率远超盘整阈值）
   - 前一段运行的起点或终点（调整区间不应包含运行段）
   - 价格振幅突然放大到与调整区间不匹配的程度
5. 统计这段调整包含多少根小时K线：
   - **≥ 90根** → 通过（S）
   - **< 90根** → 调整还不到位（C），标注当前根数，继续分析后续维度但标注"待观察"。C不代表否定，可以等多几天达到90根

### 评分

| 等级 | 条件 |
|------|------|
| S | ≥ 90根小时K线，调整充分 |
| C | < 90根，调整不到位，可继续观察等待达到90根 |

DL只看K线数量，不做其他限制。≥90即为S，<90即为C。

### 参考信息（不影响DL评分）

以下信息作为分析备注记录，供人工综合判断时参考：

- 调整之前是否有急跌/急涨（变化速率 > 0.5%/K线）
- 调整区间是否有明显倾斜（变化速率 > 0.08%/K线）

## 第二维度：平台位 (PT)

### 判断目标

在调整区间内找到价格反复验证的**关键价位**（平台位或颈线位）。这些价位被市场反复测试，证明了其作为阻力或支撑的有效性。

### 核心原则

- 平台位的本质是**关键位置、薄弱点**
- 平台位 = "中场休息搭的台子"；颈线位 = "趋势反转的头肩形态"
- 不是一条线，而是一个**区间**

### 判断规则

1. **找候选价位**：对调整区间内的高点和低点进行价格聚类

2. **硬性要求：3次以上支点测试**
   - 支撑位测试：K线最低价接近平台价，但收盘价在平台价之上（从下方弹起）
   - 阻力位测试：K线最高价接近平台价，但收盘价在平台价之下（从上方回落）
   - 不足3次测试 → 不算有效平台位

3. **测试间隔要求**
   - 两次测试之间理想间隔 **20根以上** K线
   - 多次测试最好在整个 DL 期间**比较均匀分布**，而不是集中在某一段
   - 间隔不足会降级

4. **穿越处理**
   - 实体和影线完全不穿越平台位 → 最优
   - 影线稍微超过但实体未穿越 → 可接受
   - 实体穿越（过高点） → 视为瑕疵：
     - 穿越后有≥2次有效测试 → 恢复为A（"虽然过高点，但后续有2次测试"）
     - 穿越后不足2次测试 → 直接降为C（恢复不够，平台无效）

5. **平台区间宽度**：区间范围 = 实际测试点的价格分布范围 + K线实体均宽作为上下缓冲

6. **最后一次测试后的调整要求**：DL中最后一次触碰平台位之后，价格需要先离开平台位走出一段调整结构，然后再出现TY+DN突破。不能最后一次测试完就直接突破。

### 评分规则

| 等级 | 条件 |
|------|------|
| S (优秀) | ≥ 3次测试，实体和影线均不穿越，测试间隔充分且分布均匀 |
| A (一般) | ≥ 3次测试，仅影线穿越；或实体穿越（过高点）后又有≥2次有效测试 |
| B (较差) | 多次影线穿越但仍有测试；或测试间隔不够 |
| C (不满足) | 不足3次测试；或实体穿越后不足2次测试（恢复不够，平台无效） |

### 双平台

- 阻力位（上平台）和支撑位（下平台）分别独立检测、独立评分
- **A股只检测阻力位**（见市场特殊规则），支撑位跳过
- DN 确定方向后：向上突破 → 关注阻力位；向下突破 → 关注支撑位

## 第三维度：轮廓质量 (LK)

### 判断目标

评估调整形态的"颜值"。由于价格需要反复测试平台位，调整区间天然会呈现**波浪形**。好的轮廓应该波浪规则、上下边界平滑、K线大小均匀；差的轮廓则参差不齐、忽大忽小。

### 前置检查：单边趋势

在评估轮廓质量之前，先检测结构是否为单边趋势。取结构首尾各1/4区间的收盘均价之差，除以结构总振幅（High最高 - Low最低），得到趋势比率。

- **趋势比率 > 40%** → 结构是明显的单边走势（而非盘整），LK直接判定为C，不再进行后续评估
- 趋势比率 ≤ 40% → 进入正常轮廓评估流程

### 判断规则

综合评估以下方面（偏向视觉感知，不仅仅是数学指标）：

1. **整体对称性**：调整形态的前半段和后半段是否呈对称/镜像关系
   - 好的调整像呼吸一样有节奏，前后对称
   - 从案例看："虽然中间调整幅度深，但整体走势对称和平衡" → 可以接受

2. **尾部破位检测**：调整区间的尾部是否突破了前期低点（做多方向）
   - 尾部下跌幅度过大、突破了前期低点 → 明显降分
   - 尾部走势太凌厉 → 降分但不如破位严重

3. **中间段密集度**：调整区间中间部分的K线是否紧密
   - 中间太松散、密集度不够 → 降分
   - K线均匀密集 → 加分

4. **波浪规则性**：波浪是否对称、有节奏感
   - 上轨（滚动最高价）和下轨（滚动最低价）的变化是否平缓
   - 各根K线的振幅是否接近

5. **边界平滑度**：整体轮廓的上下边界是否平整
   - 是否有突然冒出的异常大K线破坏整体形态
   - 振幅超过"均值 + 2倍标准差"的K线算异常

6. **波峰均匀性**：各波峰的高度应该相对均匀
   - 中间波峰小、两边波峰大（波峰很不均匀） → 降分到B或更低
   - 波峰不均匀且不满足头肩形态的，不能给S或A
   - 好的轮廓各波峰高度接近，或呈渐变趋势

7. **全程质量一致性**：轮廓质量需要通篇均匀，不能只看局部
   - 如果前面三分之一走势凌厉或杂乱，即使后面三分之二很好，整体评分也应降一档
   - S要求整个区间都工整；局部瑕疵会降到A

### 评分规则

| 等级 | 条件 |
|------|------|
| S (优秀) | 整体对称、波浪规则、K线密集均匀、无尾部破位、全程质量均匀、波峰高度相近、非单边趋势 |
| A (一般) | 整体尚可，可能某一方面有瑕疵（如尾部稍凌厉但整体对称；或局部走势不佳但大部分工整） |
| B (较差) | 尾部破位（突破前期低点）；或中间太松散密集度不够；或波峰很不均匀（中间小两边大） |
| C (不满足) | 多项指标不达标，形态杂乱；或单边趋势（趋势比率 > 40%） |

### 评分理由示例（来自实际案例）

- S: 形态工整，前后对称，K线密集均匀，全程质量一致
- A: "前面都挺好，尾部走势太凌厉"；"中间调整幅度深，但整体对称和平衡"；"前面三分之一LK不好，后面三分之二还不错"
- B: "尾部下跌幅度过大，突破了前期低点"；"中间太松散，密集度不够"；"中间波峰小两边波峰大，波峰很不均匀"
- C: 多项指标同时不达标；单边趋势结构（首尾偏移占振幅 > 40%）

**窄幅结构惩罚**：如果调整振幅不到均价的3%，评分阈值上调，因为过窄的调整可能是假调整。

## 第四维度：释放级别 (SF)

### 判断目标

评估调整结构的**尾部**是否向突破方向蹭上去了。好的调整应该是水平的，动能完全蓄积而不是提前释放。如果尾部向突破方向蹭了一段，说明动能已经提前消耗了一部分。

### 核心原则

- 运行 = 释放（消耗动能），调整 = 蓄力（积累动能）
- SF评估的是：这段调整是"纯粹的蓄力"，还是"蓄力的同时尾部已经在释放"？
- 因为要反复测试PT（3次以上），LK自然是波浪形，但波浪的尾部不能明显向突破方向漂移

### 判断规则

**方向相关**：SF需要结合方向来判断。做多时看尾部是否向上蹭，做空时看尾部是否向下蹭。

**方向确定规则**：
- A股：方向固定为"做多"
- 美股：**默认看多**，仅当只有支撑位达标（无有效阻力位）时才看空。同时存在阻力位和支撑位、或两者都不达标时，均按看多处理

**判断偏向视觉感知**——观察DL结构的尾部，看起来是平的、还是往突破方向蹭上去了？

| 级别 | 特征 | 含义 |
|------|------|------|
| 1st (最佳) | 调整结构整体水平，尾部没有向突破方向蹭 | 动能完全蓄积，条件满足可直接做 |
| 2nd (次优) | 尾部向突破方向蹭了一点，幅度不大 | 动能有一定消耗，需要再等一段调整才能参与 |
| 3rd (需重等) | 尾部向突破方向蹭了很多，幅度很大 | 动能已经消耗完，需要重新等待一个新的独立结构 |

### 非1st评分原因示例

- 2nd: "尾部向上蹭，幅度不大，动能有一定消耗"
- 3rd: "尾部向上蹭幅度很大，动能已消耗完"

### SF退化规则

如果尾部先向突破方向偏移（看起来像2nd），但随后出现回落K线（从尾部1/4区域的峰值到末端 ≥3根），则SF从2nd**退化为1st**。

判断标准：
1. 尾部先向突破方向蹭了一段（原本应评为2nd）
2. 在尾部1/4区域内找到峰值点（做多方向取最高收盘价）
3. 从峰值到结构末端的K线数 ≥ 3，且末端价格低于峰值 → 视为回落
4. 回落幅度须有实际意义（> 0.3% 或 > 峰值偏移的25%）→ SF = 1st

案例参考：
- 600211: "尾部3根K线回落，SF从2nd退化到1st"
- 603766: "尾部5根K线回落，SF从2nd退化到1st"
- 603909: "尾部4根K线回落，SF从2nd退化到1st"

### SF与PT位的关系

SF的偏移判断必须结合PT位的位置：

- **2nd和3rd的偏移，必须是在接近PT位的区域发生的方向性漂移**
- 如果尾部价格距离PT位较远，整段走势持续在释放中，这种释放不算作SF的"向突破方向蹭"
- 简言之：SF偏移 = 朝PT位方向的蹭动，不是泛指任何方向的价格移动
- 不接近PT位的释放走势 → 仍为1st

## 第五维度：统一区间 (TY)

### 判断目标

检测调整**最末尾**是否出现了K线越来越小、几乎不动的"挤压"状态。这是多空双方力量即将打破平衡的前兆，是最佳的入场时机信号。

### 核心原则

- 释放前的迹象：挤压到极限，筹码集中度优势
- TY必须紧贴结构末端，不能在结构中间
- 统一区间和启动K线（DN）之间最多允许1根K线间隙

### 判断规则

1. **从结构末端向前连续计数**：从DL结构的最后一根K线开始，严格向前逐根检查是否为"小K线"。如果最后1根本身不是小K线（实体 ≥ ATR × 110%），允许跳过1根继续寻找；如果最后2根都不是小K线，则TY=待定
2. **排除DN K线**：末端K线若收盘价突破PT阻力区上沿**且**K线振幅（High-Low） ≥ 小K线阈值（55% ATR），视为DN触发K线，TY计数起点从该K线之前开始。小K线（振幅 < 55% ATR）即使收盘价微超PT区间也不算DN突破——真正的DN需要有可见的力度
3. **小K线定义**（严格模式与宽松模式度量不同）：
   - **严格模式**（DN未触发/待定时）：以K线**实体**（|Close - Open|）度量。实体 < ATR × 55% 为小K线。影线不影响TY序列计数——看的是K线实体大小，不含上下影线
   - **宽松模式**（DN已确认触发时）：以K线**振幅**（High - Low）度量。振幅 < ATR × 55% 为小K线
   - 阈值55%比较严格，因为TY要求的是"视觉上明显缩小"的K线——不是看K线绝对大小，而是从完整图形来看相对大小
4. **零振幅K线处理**：振幅=0的K线为数据异常（非交易时段），不是有效K线：
   - 尾部连续零振幅K线：跳过，寻找第一根有效K线作为计数起点
   - 序列计数过程中遇到零振幅K线：视为**中断**，TY序列终止
5. **连续性要求**（分两种模式）：
   - **严格模式**（DN未触发/待定时）：TY序列必须从末端向前严格连续，遇到以下任一情况**立即中断**：
     - K线实体 ≥ ATR × 55%（实体太大，不是小K线）
     - 振幅 = 0（数据异常）
     - **跳空**：相邻K线间价格不连续（|后一根Open - 当前Close| > 小K线阈值）。TY要求价格连续性，跳空说明市场发生了跳变，不属于挤压状态
   - **宽松模式**（DN已确认触发时）：TY区域在突破K线之前，连续计数所有振幅 < ATR × 110% 的K线，遇到 ≥ ATR × 110% 中断。计数完成后验证：小K线（振幅 < 55% ATR）占比须 ≥ 40%，否则无效
6. **斜率检验**：计算挤压区收盘价的线性回归斜率，作为描述性备注记录

### 评分规则

| 等级 | 条件 | 说明 |
|------|------|------|
| 待定 | 尾部连续小K线 ≤ 2根，或尾部完全无小K线 | TY尚未形成，不做评分，但有几根就标几根 |
| C (不足) | 3根连续小K线 | 刚达到识别门槛，挤压不充分，无论斜率和K线大小如何都是C |
| B (达标) | ≥ 4根连续小K线，但斜率偏大或K线压缩程度一般 | 基本达标，挤压区已形成 |
| A (良好) | ≥ 4根连续小K线，斜率较平，K线压缩紧密 | 挤压质量好 |
| S (优秀) | ≥ 4根连续小K线，斜率接近水平，K线高度压缩 | 最佳挤压状态 |

**核心逻辑**：
- **≤2根 = 待定**，TY尚未形成，但图表上仍标注已有的小K线
- **3根 = C**，固定等级，不看其他因素
- **≥4根 = 基本达标**，进入SAB区间，由斜率和K线压缩程度（平均振幅/ATR）共同决定等级

### 案例参考

- **待定**: "完全没有"（尾部无小K线）；"最后2根，但斜率不错"（2根不够）；"只有1根有效小K线，零振幅K线中断序列"（AAPL）
- **C**: "最后3根，斜率还行"；"最后3根，倒数第4根实体太大中断了"（600573：大实体终止序列）
- **A**: "从倒数3根再往前数6根"（600292：最后2根是DN突破K线需跳过，实际TY为6根，宽松模式）；"TY有4根"（601995：从末端连续计数4根小K线，第5根实体过大中断）

## 第六维度：动能 (DN)

### 判断目标

评估是否出现了一根（或几根合并的）**有力量的突破K线**，体现某一方筹码占据压倒性优势，实质性突破了调整区间和平台位。

### 核心原则

- 好的突破是果断的：最好单根大阳线（或大阴线）一举穿越平台
- 合并K线越多，说明动能越犹豫，质量越差
- 必须是实质性突破，不是刚刚蹭到平台位边缘

### 判断规则

1. **在哪里找**：从调整结构末端开始，往后看最多10根K线
2. **突破方向**：向上和向下都检测
   - 向上：收盘价突破调整上沿，K线实体足够大
   - 向下：收盘价跌破调整下沿，K线实体足够大
3. **合并K线**：单根没突破时，尝试合并2~3根连续K线来看
4. **平台位突破确认**：
   - 基本要求：大阳线实体穿越平台位区间
   - 如果PT有瑕疵（有实体穿越但经2次测试恢复有效），DN的收盘价需要超越该穿越高点
   - 如果平台位前有小高点，最好也突破该小高点
5. **如果没有突破**：标记为"待定"

### 力度衡量

- **力度比** = 突破K线实体 / 挤压区K线平均振幅（相对于TY要有绝对力度）。如果TY=待定，则用结构尾部K线平均振幅作为基准
- **放量比** = 突破K线成交量 / 挤压区平均成交量（TY=待定时用结构尾部平均成交量）
- 整体幅度不固定，要从整个图形的视觉感知来判断是否"够强"

### 评分规则

| 等级 | 条件 |
|------|------|
| S (优秀) | 单根突破，力度强劲，实体穿越平台位，明显放量 |
| A (一般) | 单根突破，实体穿越平台位 |
| B (较差) | 2根合并突破，力度可接受 |
| C (不满足) | 3根或更多合并，或力度不足，或未穿越平台位 |

### Pending 状态

- DN还没出现突破K线时，标记为 Pending
- Pending 不代表条件差，只是还没到突破时机
- 其他五个维度的评分仍然有参考价值

## 综合展示

**不做加权打分，不做综合评级。** 六个维度的结果直接展示，由人来综合判断是否操作。

### 通用规则：非S评分必须附带原因

每个维度的评分，**除了S（或1st）以外**，A/B/C（或2nd/3rd）都必须附带文字描述原因。S级和1st级不需要特别说明。

### 结论输出格式

```
{方向}：DL{等级}/PT{等级}/LK{等级}/SF{级别}/TY{等级}/DN{等级}
仓位：{1R/0.5R/等待/不做}（原因）
```

示例：
```
看多：DLS/PTS/LKA(尾部稍凌厉)/1st/TYS/DNS
仓位：1R（所有维度均≥A）

看多：DLS/PTC(触碰不足3次)/LKB(中间松散)/2nd(尾部向上蹭)/TYA/DNS
仓位：0.5R（TY≥A+DN=S，可半仓做）

待定(多)：DLC(72根)/PTB(间隔不足)/LKA(对称性略差)/1st/TY待定(仅2根)/DN待定
仓位：等待（TY尚未形成，DN尚未触发）
```

- DL未达90根时标注当前根数，如 `DLC(72根)`
- DN未触发标注 `DN待定`，TY未形成标注 `TY待定`
- SF用1st/2nd/3rd表示
- **非S/非1st评分后面括号内附带简短原因**

## 仓位管理

仓位档位由六个维度综合决定。

### R值

- **R = 单笔最大止损金额**（当前固定1000元，未来可能改为总仓位的1%~2%）
- 买入股数 = R / (买入价 - 止损价)
- 这样不管止损幅度多大，每笔最大亏损始终控制在R以内

### 仓位档位

| 档位 | 条件 | 说明 |
|------|------|------|
| **1R** | 所有维度均在A或以上（SF=1st视为≥A） | 标准满仓位，最佳条件 |
| **0.5R** | TY≥A + DN≥S，即使PT/LK/SF不够好（甚至为C） | 半仓做，TY+DN的强势可弥补其他不足 |
| **等待** | DN尚未触发（待定）；或TY=待定；或SF=2nd+其他维度一般 | 继续观察，等待时机 |
| **不做** | SF=3rd；或关键维度不达标且TY/DN也不够好 | 条件不满足，放弃本次机会 |

### 仓位判断优先级

1. DN=待定 → **等待**（最高优先级，突破尚未发生）
2. SF=3rd → **不做**（动能已消耗完，无法挽回）
3. DL<90根 → **等待**（结构未成熟，可继续观察）
4. TY=待定 → **等待**（挤压尚未形成，继续观察）
5. 所有维度均≥A（SF=1st） → **1R**
6. TY≥A + DN≥S → **0.5R**（即使PT/LK/SF为C）
7. SF=2nd → **等待**（需再等一段调整）
8. 其余 → **不做**

## 市场特殊规则

### A股

- **只能做多**，不生成看空结论
- 如果突破方向向下 → 标注"A股不做空"
- **PT只检测阻力位（上平台），不检测支撑位** — 因为A股只做多，只关心向上突破阻力位
- SF方向固定为"做多"，只看尾部是否向上蹭

### 美股

- 可以做双边
- PT检测阻力位和支撑位，双平台独立评分
- **SF方向默认看多**，仅当只有支撑位达标（无有效阻力位）时才看空
- 主结论为确定方向，另一侧作为备注参考
