## 一、TradingView

### 产品体系

TradingView **没有公开的数据 REST/WebSocket API**，其开发者生态由以下产品构成:

| 产品 | 类型 | 面向对象 | 费用 |
|------|------|---------|------|
| **Lightweight Charts** | 开源前端图表库 (JS/TS) | 所有开发者 | 免费 (Apache 2.0) |
| **Advanced Charts** | 完整图表库 (需申请许可) | 企业/机构 | 免费许可, 需公司资质申请 |
| **Broker REST API** | REST API 规范 (券商实现) | 持牌券商 | $30,000~$50,000 初始费 |
| **嵌入式 Widgets** | iframe 组件 | 网站站长 | 免费 |
| **Pine Script + Webhook** | 策略脚本 + HTTP通知 | 平台用户 | Plus计划起 ($29.95/月) |

### Lightweight Charts
- GitHub: `tradingview/lightweight-charts`
- 功能: 折线图、面积图、K线图、柱状图
- **不包含**: 技术指标、绘图工具、交易面板
- **不提供数据**: 需自行接入数据源

### Advanced Charts (UDF 数据协议)
- 需以公司身份申请许可证, 个人开发者通常无法获得
- 定义 UDF (Universal Data Feed) HTTP 协议, 由你的后端实现:
  - `/config` - 数据源配置
  - `/history` - 历史K线 (OHLCV)
  - `/quotes` - 实时报价
  - `/search` - 品种搜索
  - `/symbols` - 品种信息
- 100+ 内置技术指标, 绘图工具, 实时数据流支持

### Webhook 告警
- Pine Script 策略触发条件时发送 HTTP POST 到指定 URL
- 延迟约 100-500ms
- 告警数量按计划限制: Plus 100个, Premium 400个, Ultimate 无限

### SDK / 语言
- JavaScript/TypeScript (npm包), React/Angular/Vue 集成示例
- Pine Script 仅在平台内运行, 不可独立部署

### 关键限制
1. **无公开数据API** - 图表库只是渲染引擎, 数据需自行解决
2. Advanced Charts 仅面向企业, 个人无法申请
3. Pine Script 不能在平台外运行
4. Webhook 需 Plus 计划起步 ($29.95/月)

## 二、富途 OpenAPI

### 架构特点
- **非传统 REST API**, 采用 **本地网关 (OpenD) + SDK** 架构
- OpenD 守护进程必须在本地或服务器运行
- 通信协议: TCP 长连接 + Protobuf/JSON 包体
- 加密: RSA 交换 AES 密钥, 后续 AES 加密

### 行情接口
| 接口 | 功能 |
|------|------|
| `subscribe` | 订阅实时数据 (报价/摆盘/逐笔/分时/K线/经纪队列) |
| `get_market_snapshot` | 市场快照 |
| `get_stock_quote` | 实时报价 |
| `get_order_book` | 买卖盘 |
| `get_cur_kline` | 最近K线 |
| `request_history_kline` | 历史K线 |
| `get_rt_ticker` | 逐笔成交 |
| `get_capital_flow` | 资金流向 |
| `get_stock_filter` | 条件选股 |
| 推送回调 | 报价/摆盘/K线/逐笔/分时实时推送 |

### 交易接口
| 接口 | 功能 |
|------|------|
| `GetAccList` | 获取账户列表 |
| `UnlockTrade` | 解锁交易 (每次交易前必须) |
| `GetFunds` | 账户资金 |
| `GetPositionList` | 持仓列表 |
| `PlaceOrder` | 下单 |
| `ModifyOrder` | 改单/撤单 |
| `GetOrderList` | 查询订单 |
| `GetHistoryOrderList` | 历史订单 |
| `GetOrderFillList` | 成交查询 |
| 推送 | 订单状态/成交回调 |

### 支持市场
| 市场 | 行情 | 交易 |
|------|------|------|
| 香港 | 支持 | 支持 (股票/ETF/窝轮/期权/期货) |
| 美国 | 支持 | 支持 (正股/ETF/期权) |
| A股 | 支持 | 支持 (通过A股通) |
| 新加坡 | 支持 | 支持 |
| 日本 | 支持 | 支持 |

### SDK
- **Python** (`futu-api`, pip安装)、**Java**、**C#**、**C++**、**JavaScript**
- 裸协议 (Protobuf) 支持任意语言自行实现

### 费用
- API 使用: **免费**
- 交易佣金: 与APP一致, 无额外收费
- 港股 LV2 行情: 大陆IP免费
- 美股行情: 部分需购买行情卡

### 频率限制 (按用户等级)
| 用户等级 | 订阅额度 | 历史K线额度 | 条件 |
|---------|---------|-----------|------|
| 标准 | 100 | 100 | 已开户 |
| 二级 | 300 | 300 | 资产 >= 1万港元 |
| 三级 | 1,000 | 1,000 | 资产 >= 50万港元 |
| 四级 | 2,000 | 2,000 | 资产 >= 500万港元 |

### 关键限制
1. 必须运行 OpenD 网关 (非云端API)
2. 必须在富途/moomoo APP 开户
3. 行情权限与APP端不完全共用
4. 交易前每次需解锁 (输入交易密码)

## 三、老虎证券 Open API

### API 类型
| 类型 | 用途 |
|------|------|
| **REST API** | 交易下单、账户查询、行情快照 |
| **WebSocket (STOMP)** | 实时行情订阅、订单/持仓/资产变动推送 |

### 认证方式
- **RSA 非对称签名**: 生成密钥对, 公钥上传平台, 私钥本地签名请求
- 核心参数: `tiger_id` + `timestamp` + `sign`
- SDK 已封装签名逻辑

### 行情接口
- 实时/延迟报价、K线 (分钟~年线)、分时数据
- 深度行情 (美股40档, 港股10档+经纪商队列)
- 逐笔成交 (Tick)、资金流向、行情扫描器
- 单次请求上限: 50只标的

### 交易接口
- 下单/改单/撤单
- 订单类型: 市价/限价/止损/止损限价/跟踪止损/附加订单
- 算法订单: TWAP、VWAP
- 支持盘前盘后交易 (Prime账户)

### 支持市场
| 市场 | 品种 |
|------|------|
| 美股 | 股票、ETF、期权、期货 (含碎股) |
| 港股 | 股票、ETF、期权、期货、窝轮、牛熊证 |
| A股 | 股票 (Prime+模拟) |
| 新加坡 | 股票、ETF、期货 |
| 外汇 | 外汇交易 |

### SDK
- **Python** (`tigeropen`)、**Java**、**C#**、**C++**

### 频率限制 (60秒滚动窗口)
| 等级 | 限制 | 接口示例 |
|------|------|---------|
| 高频 | 120次/分 | 实时行情、下单 |
| 中频 | 60次/分 | K线、账户查询 |
| 低频 | 10次/分 | 市场状态、Token刷新 |

### 费用
- API 使用: **免费** (开户+入金)
- 行情: **独立于APP, 需单独购买**
- 交易佣金: 与APP一致

### 关键限制
1. 行情权限与APP完全独立, 需单独购买
2. 超频可能被列入黑名单
3. Global账户需联系客服开通API

## 四、同花顺

### 4.1 iFinD 数据接口

面向量化研究的金融数据 API (类似Wind API):

| 函数 | 功能 |
|------|------|
| `THS_HF()` | 高频数据 (分钟线/Tick) |
| `THS_RQ()` | 实时行情快照 |
| `THS_HS()` | 历史行情 (日/周/月线) |
| `THS_BD()` | 基础数据 (财务/公司信息) |
| `THS_DS()` | 时间序列数据 |
| `THS_DR()` | 数据池 (板块/指数成分) |
| `THS_EDB()` | 宏观经济数据 |
| `THS_WC()` | 智能选股 (i问财) |
| `THS_ReportQuery()` | 公告查询 |

**认证**:
- SDK: 账号密码登录 `THS_iFinDLogin()`, 绑定单台设备
- HTTP: `refresh_token` + `access_token` 双令牌, 绑定最多20个IP

**SDK**: Windows (Python/Matlab/R/VBA/C++/C#/Java), Linux (Python/C++/Java), HTTP (任意语言)

**调用限额**: 行情 1.5亿条/周, 基础数据 500万条/周

**价格**: 约 **0.88~2.80万元/年**

**限制**: 无交易能力 (纯数据), 美股延时行情, SDK绑定单设备

### 4.2 SuperMind 量化平台 (原MindGo)

在线量化策略平台, 提供回测+仿真+实盘:

| 版本 | 价格 | 功能 |
|------|------|------|
| 模拟体验版 | **免费** | 无限回测、LV2数据 |
| 机构量化版 | **6万/年** | 实盘交易 |

- 仅支持 Python
- 覆盖: A股、基金、可转债、融资融券、期货
- **不支持期权**
- 策略运行在同花顺服务器端, 不绑定特定券商

## 五、长城证券

### 量化工具

| 工具 | 说明 | 门槛 |
|------|------|------|
| **GWT 策略交易系统** | 自有系统, Python/C++, 本地运行 | 主要面向机构 |

- **xtdata 行情模块**: K线/分笔/实时行情/财务数据/板块数据
- **xttrader 交易模块**: 下单/撤单/资金查询/持仓查询/回调推送

### 覆盖市场
- A股 (沪深)、港股通、期货、期权

### 费用
- QMT/miniQMT 免费开通 (满足资金门槛), 按佣金收费

## 选型建议

### 按使用场景

| 场景 | 推荐 | 理由 |
|------|------|------|
| **纯数据获取** (研究/回测) | 同花顺 iFinD | 数据全面, 价格低于Wind |
| **A股量化交易** (个人) | 国金证券 (miniQMT) | 门槛低, 工具全, 文档好 |
| **港美股量化交易** | 富途 OpenAPI / 老虎证券 | 多市场覆盖, SDK成熟 |
| **Web图表展示** | TradingView Lightweight Charts | 免费开源, 轻量高性能 |
| **一站式量化平台** | 同花顺 SuperMind | 免费回测, 不绑券商 |

### 按 ZQ-Trade 项目对接优先级

考虑到项目以 A股+港股+美股 K线分析为核心:

1. **国金证券 miniQMT** - A股交易执行首选, 门槛10万, xtquant Python API 可直接集成
2. **富途 OpenAPI** - 港美股交易执行, 多市场覆盖, Python SDK 成熟
3. **老虎证券** - 港美股备选, REST API 架构更适合云端部署
4. **同花顺 iFinD** - 如需更丰富的基础数据 (财务/宏观) 可补充
